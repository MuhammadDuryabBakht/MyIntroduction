# 🌐 Personal Portfolio Website (HTML)

This repository contains my **first HTML project**—a simple multi-page personal portfolio website built using **HTML5**. The project showcases my introduction, educational background, and professional experience while helping me strengthen my understanding of the fundamentals of web development.

## 📖 About the Project

The website consists of three interconnected pages:

- **Home Page** – A brief introduction about me with a link to my LinkedIn profile.
- **Education** – My academic journey from school to university.
- **Experience** – My professional experience as a Quality Control Supervisor at Naveed Enterprises.

This project focuses solely on HTML and does not use CSS or JavaScript. It emphasizes proper webpage structure and semantic HTML elements.
